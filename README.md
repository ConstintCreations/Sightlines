# Sightlines

**Sightlines** minimalist logic puzzle web game about vision, heavily inspired by [0h n0](https://0hn0.com/) by [Q42](https://www.q42.nl/en/). 

## How to Play
- Goal:
    - The goal of the game is to fill the grid as fast as possible while satisfying the rules.
- Rules:
    1. Each **numbered blue cell** must *see* exactly that many blue cells in total (not including itself) across all four direction, until blocked by a **red cell**
    2. No cell may remain empty
    3. Every **unnumbered blue cells** must be seen by at least 1 **numbered blue cell** 
- Controls:
    - Left Click: Cycle a cell between Blue (vision) -> Red (blocker) -> Gray (empty)
    - Right Click: Cycle in reverse
